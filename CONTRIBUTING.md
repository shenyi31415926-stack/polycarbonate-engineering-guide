# Contributing to Polycarbonate Engineering Guide

## How to Contribute

### Reporting Issues
- Use the GitHub issue tracker
- Include technical specifications when relevant
- Provide clear reproduction steps for calculation errors

### Submitting Changes
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-content`)
3. Commit your changes (`git commit -m 'Add: technical specification for XYZ'`)
4. Push to the branch (`git push origin feature/amazing-content`)
5. Open a Pull Request

### Content Standards

All contributions must:
- Include technical data with sources
- Use metric units (SI) with imperial equivalents in parentheses
- Include at least one calculation example
- Follow Markdown formatting conventions
- Be technically accurate and verifiable

### Review Process
- Technical accuracy review
- Formatting compliance check
- Integration testing (for calculators/tools)
